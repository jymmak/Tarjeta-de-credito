             var creditCard = prompt("Ingrese el número de su tarjeta");
	     creditCard!== "" && creditCard !== " "

        4.- isValidCard(numbersCard) , esta función verificará si los números de la tarjeta de crédito son válidos al Algoritmo de Luhn.
	5.- var arr = [];    creamos un array
	6.- var size = numbersCard.length;   nos da el tamaño en número de caracteres.
	7.- var lastPosition = size - 1;     nos da la última posición.
	8.- var sumtotal = 0;

	9.- si : i>=0  for(var i = lastPosition ; i >= 0 ; i--)
		arr.push(parseInt(numbersCard[i])); guarda el  número impar.
	
	10.- Si : var j=1 y j <= arr.length, continúa
		(j % 2) === 0 si es verdad continúa si es falso te envía a j++: 
			var number = arr[j-1] * 2;    guarda el  número par, continúa          
			number >= 10 , en caso que multiplicación sea mayor igual a 10.
				si es verdad entonces number = number.toString();

				luego segundo dígito : var digitTwo = parseInt(number[1]);  number sera un número.
				continúa con: arr[j-1] = digitOne + digitTwo;  de más dígitos terminando en j++.
		Si number >= 10 es  falso nos da arr[j-1] = number terminando en j++ regresando a j<=arr.length donde j sea  menor que la longitud del array "arr",
	11.-  Si j<=arr.length es false creamos una nueva variable  llamada  "k" var k = 0 ; k < arr.length si es verdad
		sumtotal = sumtotal + arr[k] ;  Sumatoria de todos los numeros de la tarjeta.
